# 🎨 Interactive Color Changer

A simple, interactive web application that changes the background color of a box randomly when a button is clicked. Perfect for demonstrating DOM manipulation and event handling in JavaScript.


## ✨ Features

- 🎲 **Random Color Generation** - Generates random hexadecimal colors
- 🖱️ **Interactive Button** - Click to change the box color
- 🎨 **Smooth Transitions** - CSS animations for better user experience
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile devices
- 💻 **Real-time Updates** - Instant color changes with visual feedback


## 🛠️ Technologies Used


| HTML5 | Structure and semantics |
| CSS3 | Styling, animations, and responsive design |
| JavaScript (ES6) | DOM manipulation and event handling |
| Git | Version control |
